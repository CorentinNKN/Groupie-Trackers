# Groupie-Trackers
## Presentation du Projet
Ce projet consiste en la creation d'une application de visualisation de donnees pour des groupes de musique. L'application consomme une API REST composee de quatre parties principales : les artistes, les lieux de concerts, les dates de concerts et les relations entre ces donnees. L'interface a ete realisee en langage Go avec le framework Fyne, en respectant les principes d'ergonomie de Schneiderman.
